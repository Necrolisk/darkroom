# darkroom

* Create a map of tuples representing x, y coordinates of n size.

* Randomly assign a exit, a monster position & a player position.

* The player wins if they reach the exit, losees if they land on the grue.

######Example size 4 grid:
```
(0, 0), (0, 1), (0, 2), (0, 3)
(1, 0), (1, 1), (1, 2), (1, 3)
(2, 0), (2, 1), (2, 2), (2, 3)
(3, 0), (3, 1), (3, 2), (3, 3)
```
